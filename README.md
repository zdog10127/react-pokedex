<!-- GETTING STARTED -->

- npm

```sh
npm install npm@latest -g
```

### Installation

1. Clone the repo

```sh
git clone https://github.com/ShinteiMai/react-pokedex.git
```

2. Install NPM packages

```sh
yarn
```

3. Run the local development server
